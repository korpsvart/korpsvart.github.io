Download ed esecuzione del programma



Per eseguire il programma, è necessario:

  • Disporre sulla propria macchina della JRE (Java Runtime Environment) aggiornata almeno alla versione 11
  • Scaricare ed eseguire il file JAR usando il pulsante sottostante e successivamente:
    • Fare doppio clic sul file
    • Lanciare il seguente comando da riga di comando: java -jar RayTracer.jar
